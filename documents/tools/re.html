Random Entity Generator

by pablomcprojects

Properties

Variant Number:   

Textures

Number or number range of the textures this variant uses.

Weights

The chance each texture has of displaying in this variant, the number of weights must match the same number of textures used.

Heights

Specify the height (Y layer) number or number range the this variant will display in.

Name

The variant will only display in entities that match this name.

Biomes

Specify the biomes this variant will display in.

Health

The variant will only display in entities that match the specified health, health range, or health percentages.

Weather

The variant will only display when the weather is:
 -clear: when is neither rain or a storm going on.
 -rain: when its raining.
 -thunder: when there's a storm going on.

Time of the day

The variant will only display within the specified tick range of a day in Minecraft.

Moon Phase

The variant will only display within the specified moon phases (0-7).

Baby

The variant will only display when the entity is a baby.

Collar Colors

The variant will only display when the collar matches the specified color(s).
