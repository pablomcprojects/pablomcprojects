CIT Generator

General Properties

Type

Type of texture to replace:
 -item: simple item texture replacement (held items, GUI, dropped, etc).
 -enchantment: overlay texture for enchanted items (glint).
 -armor: armor models worn by players and mobs.
 -elytra: simple item texture replacement (held items, GUI, dropped, etc).

Item

Namespaced ID of the item to apply the texture to.

Name

The item display name that has to match for the texture to apply.

Description

The item description that has to match for the texture to apply.

Type Specific Properties

Blending mode

Blend method when overlapping textures.
 -add: adds pixel values of one layer with the other.
 -subtract: subtracts pixel values of one layer with the other.
 -multiply: multiplies the pixel values of one layer with the other.
 -dodge: sums the pixel values of the layers.
 -burn: sums the pixel values of the layers and subtracts 1.
 -screen: inverts layers, multiplies them, and then inverts that result.
 -replace: replaces the previous layer with the specified one.
 -overlay: combines multiply and screen blend modes.
 -alpha: weighted average by alpha value.

Speed

Scrolling speed of texture, default speed is 0 (no scrolling).

Rotation

Angle of texture relative to the item, default speed is 0.
If speed is non-zero, the texture will also scroll in this direction.

Layer

Specifies a unique layer and the ordering of the layers as they overlap each other.
If they use the same layer, weight determines which one renders. Default is 0.

Duration

Duration in seconds of the enchantment in a cycle, default is 1 second.
(Only for cycling enchantments)

Path Properties

Texture

Path to the replacement texture. It can be a full path or just the name.

Model

Path to the replacement model. It can be a full path or just the name.

Enchantment Properties

Enchantment

List of enchantments the item has to match for the replacement texture to apply.

Enchantment level

Levels for the enchantments specified before.

Miscellaneous Properties

Hand

Specifies which hand the item is rendered:
 -any: matches items held in any hand + inventory.
 -main: only matches items held on the main hand (also applies to GUI).
 -off: matches items not held with any hand.

Stack size

Size of the stacked items (usually either 1, 16 or 64).

Weight

If multiple files match the same item, the one with highest weight is used.
If type=enchantment it also determines the rendered layer for the glint.

Damage taken

For items with durability, damage starts at 0 and increases as it gets damaged.
For other items, damage represents different properties (potion type or wool color).

Potion Properties

Potion

Only if item=minecraft:potion. Potion to apply the texture to.
